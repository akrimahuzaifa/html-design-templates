# HTML Design Templates

A small collection of simple HTML/CSS templates and components for quick prototyping and visual reference.

Contents
 - `alpha-footer-signature` — footer templates ([open](alpha-footer-signature/index.html)).
 - `beaver-creak` — single-page example ([open](beaver-creak/index.html)).
 - `byte-werk-3d` — 3D effect demo with assets ([open](byte-werk-3d/index.html)).
 - `day-night-toggle-btn` — CSS toggle button demo ([open](day-night-toggle-btn/index.html)).
 - `portfolio gtmatrix` — small notes/text file for portfolio ([open](portfolio%20gtmatrix/porfolio.txt)).
 - `testiminials-cards` — testimonial card variations ([open](testiminials-cards/testimonial.html)).

Quick preview

Open any of the included `index.html` files directly in your browser, or serve the folder and browse to the file. Example (from repository root):

```bash
# serve on port 8000 using Python 3
python -m http.server 8000
# then open in browser, e.g. http://localhost:8000/alpha-footer-signature/index.html
```

Recommended workflow

 - Use VS Code + Live Server for quick previews.
 - Edit the HTML/CSS files in-place and refresh the browser to see changes.
